# Attack on Titan: Utopia

## Overview

Attack on Titan: Utopia is a Java-based tower defense game inspired by the Attack on Titan universe. The project was developed as part of the Object-Oriented Programming course at the German University in Cairo.

## Technologies

- Java
- Eclipse
- Object-Oriented Programming

## Features

- Multiple Titan types
- Different weapon classes
- Battle phases
- Lane management
- Resource management
- Game engine implementation
- Exception handling

## OOP Concepts Applied

- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Interfaces
- Design Patterns

## Project Structure

- game.gui
- game.tests
